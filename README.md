# arthurcrodri.github.io

## 📝 | Resumo do Projeto

Repositório destinado a armazenar o site do perfil de Arthur C. O. Rodrigues, com informações acadêmicas, profissionais, experiências e competências. Desenvolvido com base nas aulas do curso Discover da RocketSeat e inicialmente desenvolvido como um desafio do Processo Seletivo da Include Engenharia.

## 💻 | Tecnologias

Este projeto utiliza:

- HTML;
- CSS;
- JavaScript;
- Git, GitHub e hospedagem com GitHub Pages.

## 🚀 | Aprendizados

Entre os principais aprendizados que tive com este projeto, estão a estilização de elementos com CSS, visto que não sabia utilizar tal tecnologia até o desenvolvimento do site, assim como o uso de `localStorage` em JavaScript para manter o tema selecionado pelo usuário (claro ou escuro) ao navegar entre as páginas.

## 👨‍💻 | Bugs, Glitches e *Issues* Conhecidas

- O site não está inteiramente otimizado para telas com orientação vertical;
- As transições podem não funcionar em alguns navegadores.
