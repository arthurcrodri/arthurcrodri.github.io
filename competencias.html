Competências e Habilidades

Python

Tem contato com a linguagem desde 2022, sabendo criação e manipulação de arrays, estruturas de condição e repetição, interpretação do input do usuário, etc. Um projeto desenvolvido foi um rolador de dados de RPG em linha de comando (PyDiceRoller), que recebia o tipo de dado que o usuário queria rolar, comparava o input com uma array de possibilidades e, caso fosse uma opção válida, o programa retornava um número aleatório de acordo com o número de faces do dado (d4 = um número de 1 a 4, d20 = um número de 1 a 20...)

HTML

Possui domínio da linguagem de marcação, podendo desenvolver websites com facilidade utilizando esta tecnologia, com aprendizado desde 2021. Um exemplo é este website, inteiramente construído usando HTML5.

CSS

Possui domínio intermediário de estilização de sites HTML, com conhecimento das formas de posicionamento, atribuição de imagens, SVGs, dimensionamento, posicionamento de elementos, etc. Este website, por exemplo, foi inteiramente estilizado utilizando CSS.

GitHub

Possui conhecmento básico de GitHub, tendo usado pela primeira vez em 2024. Sabe criar e organizar repositórios, fazer pull requests, publicação e push de arquivos modificados, bem como a utilização do GitHub Pages para hospedar os sites na rede. Este site, por exemplo, está hospedado via GitHub Pages e seu repositório está disponível aqui.

Pacote Office
